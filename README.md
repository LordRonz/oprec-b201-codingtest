# Open Recruitment B201Lab 2022

Repositori yang berisi problem-problem yang digunakan untuk tes coding oprec b201lab. Di tiap folder terdapat solution yang berupa kode C/C++. Lalu ada juga kode python `generate_testcase.py` untuk membuat testcase secara otomatis. Di folder testcase terdapat file-file input dan output dari testcase yang digunakan saat hari-H pelaksanaan tes coding. Lalu terdapat pula file markdown `README.md` yang adalah deskripsi dari masing-masing problem. Pada root path terdapat shell script `verify_solutions.sh` yang digunakan untuk melakukan cek apakah solution.c/cpp sudah benar atau belum dengan membandingkan input dan output dari program dengan testcase. Lalu untuk melakukan pembuatan seluruh testcase dapat dieksekusi file `generate_testcase.sh`.
